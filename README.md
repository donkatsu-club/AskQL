# 🔎 AskQL
</br>

## 1️⃣ 프로젝트 개요

AskQL은 **SQL 정규표현식**을 공부할 때 유용한 학습 도우미이다. 고객의 카드 사용 패턴 데이터를 활용한 문제를 제공하여 학습 내용에 대한 이해도를 평가하고, 실력을 단계적으로 향상시키는 것을 목표로 한다.
