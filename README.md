## 1️⃣ 프로젝트 개요

**AskQL**은 **SQL**과 **정규표현식** 학습을 돕는 도구입니다.

**첫째**, 고객 카드 사용 패턴 데이터를 활용한 다양한 문제를 제공합니다.

**둘째**, 학습자의 이해도를 체계적으로 평가합니다.

**셋째**, 단계별로 실력을 향상시킬 수 있도록 지원합니다.
